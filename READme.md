# Temperature-Controlled-Fan-Using-8051-Microcontroller
The aim of the project is to control the speed of a Fan (DC Motor) according to the 
temperature of the environment. The more the temperature, the more will be the speed of 
the fan and maintains the coolness of the surroundings. The fan doesn't switch on for 
temperature below 0°C and slowly varies its speed from minimum to maximum over 0 to 
150°C. The circuit is simulated in Proteus Simulation Software.
